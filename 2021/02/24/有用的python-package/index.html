Numba–矩阵运算加速

注意：在numba中使用一个普通的python列表不是一个好主意，因为它将花费很长时间来验证类型。

使用ndarray，才是正确的方法！才能带来速度的提升。

另外，@vectorize可以将处理一个元素的函数，转换成可以接受 array输入的优化函数，只是第一次使用需要对内存分配进行优化，会慢一些。
